## Tech

- [Angular](https://angular.io/)
- [Angular Material](https://material.angular.io/)
- [Chart.js](https://www.chartjs.org/)

## Installation

Requirement Angular CLI

```sh
npm install -g @angular/cli
```

Install the dependencies and start the server

```sh
npm i
npm run dev
```
